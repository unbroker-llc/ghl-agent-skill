# SaaS API

API Service for SaaS

Version: 1.0

## Base URL

- https://services.leadconnectorhq.com

## Authentication

- **bearer**: http (bearer) [JWT] — Use the Access Token generated by exchanging API Key for token id
- **Location-Access**: http (bearer) [JWT] — Use the Access Token generated with user type as Location
- **Agency-Access**: http (bearer) [JWT] — Use the Access Token generated with user type as Company
